# 📱 Travel App PWA

A Progressive Web Application built using **Angular + Ionic** that allows users to explore tours, view details, save favorites, and make bookings.

---

## 🚀 Features

- Browse a list of available tours with images  
- View detailed information for each tour  
- Add and remove tours from favorites  
- Save user location using device geolocation  
- Book tours using a simple form  
- Persistent storage using Ionic Storage  

---

## 🛠 Technologies Used

- Angular (Standalone Components)
- Ionic Framework
- Capacitor (Geolocation plugin)
- HttpClient (API data)
- Ionic Storage (Local data persistence)

---

## 📦 Installation & Run

Clone the repository and install dependencies:

```bash
npm install
ionic serve
```
The app will open in your browser.
